Help

  • This is an advanced phishing detector. If you believe a website is trying to steal your information, copy its URL and paste it in our website, then click 'Check URL'.
  • The URL has to start with the protocol, e.g., https://canvas.bham.ac.uk/. If the URL is invalid, a message will appear on the screen.
  • The results of the tests will be displayed on the lower side of the home screen. The analysis process may take up to 3 minutes, but will likely take less than one minute.
  • Our website is using four phishing detection methods. This is what makes it so accurate. The results are separately displayed on the main page after clicking 'Check URL'.
  • The first method used consists of checking if clicking the link triggers a file download. In this case, our software checks whether that file is malicious or not. If it is, our website displays an informative message, and the website is directly flagged as phishing. If not, the website is considered safe. If the URL does not trigger any download, our software will use the other 3 methods.
  • The second phishing detection method is a machine learning model trained and tested using over 11000 URLs. Each submitted website is analysed and divided into 26 attributes, such as whether the URL uses an IP address, URL length, utilisation of a URL shortening service, etc. Based on these attributes, the model is able to predict whether the website is a phishing attempt or not.
  • The third method consists of a content-based filtering algorithm. This method is only available for English websites, so check the 'Website in English' box if this is the case. It looks for the use of urgency language, trust cues and spelling errors. Although many attackers use this language to scare the people and persuade them, this may not always be the case, so this test might not always be useful. But keep in mind: if the website has words from other languages, it will likely be labelled as phishing, even though that might not be the case.
  • The fourth and last method checks 4 constantly updated blacklists from the internet and displays the result.
  • In the top-left corner, there is the 'History' section. You can find there all the websites that were checked by our users, together with the results. The results are in the same order as they appeared on the home screen: machine learning model, language check (if the 'Website in English' checkbox is checked) and blacklists, in this order. In case the URL triggers a file download, there will be only the result of this test, together with a suggestive message. Under the history table you can find the translation of the history abbreviations.
  • Our 'History' has a search bar on the top of the page. In case you want to see whether a certain website was checked using our detector, you can type a keyword, click 'Search', and the page will display the corresponding results.
  • In the top-right corner, you can find the 'Help' section.
  • We released a phishing detection extension for Google Chrome as well. After installing the extension in your Chrome browser, click the round blue icon from the upper-right corner whenever you are browsing a suspect website. Click one of the two buttons, depending on the language of the website, and wait for the results to be displayed. The extension uses the same phishing detection methods as our software, except the malicious downloads one.
